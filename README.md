## Featured Project

### [E-commerce Data Cleaning & Visualization](https://github.com/Titiksha-P/ecommerce-data-cleaning-visualization)

<a href="https://github.com/Titiksha-P/ecommerce-data-cleaning-visualization">
  <img src="https://raw.githubusercontent.com/Titiksha-P/ecommerce-data-cleaning-visualization/main/sales_by_category.png" width="700" alt="E-commerce data cleaning and visualization project">
</a>

An end-to-end data analysis project that transforms a deliberately messy e-commerce sales dataset into clean, structured, and interpretable business insights.

### What I Implemented

* Identified and removed duplicate records
* Handled missing values using appropriate imputation methods
* Standardized inconsistent city, category, and payment labels
* Corrected impossible customer-age values
* Treated numerical outliers using the IQR method
* Engineered gross sales, net sales, order-month, and age-group features
* Created visual reports using Matplotlib
* Developed an Excel dashboard containing summary metrics and charts

### Project Results

* Cleaned **332 raw records** into **320 reliable records**
* Identified **Clothing** as the highest-sales category
* Identified **Pune** as the strongest city market
* Found **June 2025** to be the highest-sales month
* Calculated total net sales of approximately **₹237,342**

**Technologies:** `Python` `Pandas` `Matplotlib` `Jupyter Notebook` `Excel`
